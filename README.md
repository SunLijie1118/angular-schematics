# angular schematics

本例中用angular schematics实现了ng generator使用模板生成组件、ng add指令安装模块、ng update更新模块修改指定组件内容

## 修改并使用

```bash
npm run install
npm run build
npm link
```
### 实现ng generator使用模板生成组件
使用示例
```bash
ng g angular-schematics:component 
```
### 实现ng add指令安装模块
使用示例
```bash
ng g angular-schematics:ng-add 
```
### 实现ng update更新模块修改指定组件内容
使用示例
```bash
ng update
```
